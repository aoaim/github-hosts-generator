# GitHub520

在本地生成最优 GitHub hosts 配置，解决访问慢、图片加载失败等问题。

> 本项目基于 [521xueweihan/GitHub520](https://github.com/521xueweihan/GitHub520) 改进，将 ICMP ping 改为 TCP 连接测试，无需 root 权限即可运行。
